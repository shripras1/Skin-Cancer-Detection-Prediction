# Malignancy Detection in Skin Cancer Image Data

## Project Overview - TO-DO
This is a comparative study of different complex deep learning architectures for malignant lesion detection using various skin sample images. 

## Dataset and Setup Instructions <br>

### Where and How to Download
The data can be found at Kaggle: [Skin Cancer Detection](https://www.kaggle.com/competitions/isic-2024-challenge/data)
<br>
Because of the sheer size of the data, the code notebooks largely used Google Colab, accessing the data om Kaggle using a generated API Token.
1. On your Kaggle account, create a Legacy Key and an API Token as needed, copying both
    * When generating the Legacy Key, a kaggle.json file is downloaded onto the local machine being used. Know the location of this file
    * When gererating the API Token, copy and note down the token -- it will not be displayed again
2. When asked for credientials in the coding notebooks:
    * When prompted to upload files by Colab, upload the relevant kaggle.json file to the notebook
    * When asked to verify Kaggle credientials, paste the generated token in the textbox

### How to Run Each Code File 
Follow the steps to access the Kaggle dataset through Google Colab, and the provided code notebooks should run without any errors

### Data Preprocessing Steps How To Train and Run Each Model, and Expected OutPuts 
Each code notebook (with code on respective models) should be able to run in sequence, completing any data cleaning and preprocessing without throwing any errors, upon following the steps to access the data through Google Colab.
